Project Name

Day & Night Button Theme

Objective

To design and test a modern light/dark mode toggle button with smooth animation for better accessibility and user experience.

User Research

Target Users:

  • General users who frequently switch between light and dark modes on websites/apps.

Pain Points Identified:

  • Default toggles feel plain and lack engaging animation.
  • Users want instant feedback when switching modes.
  • Accessibility issues: buttons are sometimes too small or not visible in both themes.

User Goals:

  • Easily switch between themes.
  • Visually appealing toggle animation.
  • Clear indication of current mode.

Project Name

Image Detection With ML

Objective

To design and test a mobile UI that allows farmers, researchers, and agricultural workers to upload crop images, classify them into predefined categories, and detect diseases using machine learning. The app focuses on simplicity, accessibility, and real-time feedback.

User Research

Target Users:
Farmers with limited technical knowledge, agricultural researchers, students and learners in agriculture & plant pathology.

Pain Points Identified:

  • Farmers face difficulties identifying crop diseases.
  • Most existing tools are complex and not mobile-friendly.
  • Lack of localized solutions tailored for specific crops (e.g., dragon fruit).

User Goals:

  • Upload or capture crop images easily.
  • Select or add crop classes dynamically.
  • Get fast detection results with visual feedback.
  • Simple, clear UI without technical jargon.

UI Design Features

  1. Onboarding Screen (Logo + Branding): Friendly mascot (dragon fruit icon), gradient background, clear app identity.
  2. Class Input Screen (Data Setup): Input fields to add crop categories, dropdown for ‚ÄúAdd More Class‚Äù, continue button.
  3. Processing Screen (ML Detection): Loading bar to show progress, background crop image for branding.

Prototype & Workflow

  1. User logs in ‚Üí lands on Home screen.
  2. User adds/selects crop classes.
  3. Uploads/captures an image.
  4. App shows Processing screen with progress indicator.
  5. Output: Detected class/disease category displayed.

Tool Used: Figma
Prototype Link: Open Prototype

Impact & Use Cases

  • Helps farmers detect crop diseases early, reducing yield loss.
  • Supports agricultural researchers with labeled datasets.
  • Can be extended to multiple crops beyond dragon fruit.
  • Provides AI + UI integration for real-world agricultural solutions.
